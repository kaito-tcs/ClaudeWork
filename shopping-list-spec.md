# おかいものリスト — 仕様指示書

## 概要

ブラウザで動作するシングルページの買い物リストアプリ。  
ローカルストレージにデータを保存し、サーバー不要で利用できる。

---

## ファイル構成

```
shopping-list.html   HTMLマークアップ（構造のみ）
shopping-list.css    スタイル定義
shopping-list.js     ロジック・データ管理
```

---

## 機能一覧

### リストビュー（🛒 リスト）

| 機能 | 詳細 |
|------|------|
| 商品追加 | 商品名・数量・カテゴリを入力して追加。Enterキー対応 |
| 完了トグル | アイテムをクリックすると購入記録モーダルを開く |
| 削除 | ホバーで削除ボタン表示。スライドアウトアニメーション付き |
| カテゴリフィルター | タブでカテゴリ絞り込み |
| 進捗バー | 完了数 / 全体数をリアルタイム表示 |

### カテゴリ

| キー | 表示名 | カラー |
|------|--------|--------|
| `veggie` | 🥦 野菜・果物 | `#5B8A4A` |
| `meat`   | 🥩 肉・魚     | `#C4622D` |
| `drink`  | 🥛 飲み物・乳製品 | `#3A7BD5` |
| `bread`  | 🍞 パン・穀物 | `#8B6B4A` |
| `other`  | 🛒 その他     | `#7A6B8A` |

### 購入記録モーダル

- アイテムを完了にする際にボトムシートで表示
- 入力項目：**金額（円）**・**購入場所**（どちらも任意）
- ボタン：キャンセル / スキップ（記録なしで完了）/ 記録して完了

### 履歴・グラフビュー（📊 履歴・グラフ）

- 購入履歴を商品名でグループ化して表示
- 最終購入日が新しい順にソート
- 各カードに以下を表示：
  - 平均・最安・最高価格のサマリー
  - 価格推移の SVG ラインチャート
  - 最新6件の購入履歴テーブル（日付・金額・場所）

## データ構造（localStorage）

### `shoppingItems`

```json
[
  {
    "id": 1746089400000,
    "name": "トマト",
    "qty": 2,
    "cat": "veggie",
    "done": false
  }
]
```

### `purchaseHistory`

```json
[
  {
    "id": 1746089500000,
    "itemName": "トマト",
    "category": "veggie",
    "price": 128,
    "location": "スーパー田中",
    "date": "2026-05-01T08:00:00.000Z"
  }
]
```

## 主要関数（shopping-list.js）

| 関数 | 役割 |
|------|------|
| `addItem()` | フォームから新規アイテムを追加 |
| `toggleItem(id)` | 未完了→完了時にモーダルを開く。完了済み→未完了は即時トグル |
| `confirmRecord()` | モーダルの入力を購入履歴に保存してアイテムを完了にする |
| `skipRecord()` | 記録なしでアイテムを完了にする |
| `deleteItem(id)` | アニメーション付きでアイテムを削除 |
| `renderList()` | リストビューを再描画 |
| `renderHistory()` | 履歴ビューを再描画 |
| `renderChart(priceRecords, color, idx)` | SVG ラインチャートを生成して返す |
| `setFilter(cat)` | カテゴリフィルターを切り替え |
| `switchView(view)` | リスト / 履歴ビューを切り替え |

## デザイン方針

- **カラーパレット**：ウォームクリーム背景 (`#F5EFE3`)、フォレストグリーン主色 (`#2C4A3E`)
- **フォント**：見出しに `Fraunces`（英語 / イタリック）、本文に `Noto Sans JP`
- **アニメーション**：追加時 `itemIn`、削除時 `itemOut`、モーダルはボトムシートスライドイン
- **レスポンシブ**：460px 以下でフォームを縦並びに変更
